Synchronize tmux panes

2016-06-22 by fredrik

So we have clustered servers which is a pain when you want to look at some log files. tmux and stackoverflow to the rescue. I set up tmuxinator to open up panes on all the servers and then once inside tmux I go:

C-B :
setw synchronize-panes yes

…and then I can tail and search all the logs at the same time. (This obviously only works when servers are set up in the same way…).
